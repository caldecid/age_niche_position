#Age-Dependent Niche Position

This is a repository with the R scripts and processed data for a research paper, currently under review. 

Inside the "results/scripts" folders there are three folders which contain the methodological workflow of the paper:

* "ENFA" which contains the scripts related to the ENFA analyses
* "functions" which contains the scripts with the general functions used to run the analyses.
* "phylo_manipulation" which contains the scripts with the phylogenetic manipulations and Phylogenetic generalized least squares (PGLS) analyses.

Inside the "results/data" folders there are two folders which contain the data generated during the analyses.

* "metada" which contains the data generated during the ENFA analyses
* "processed" which contains the data generated during the phylogenetic manipulations and the results from the PGLS.

The instructions for running the code are not very detailed, but they are explicit enough for evaluating our research workflow.
